About CamelTutor

CamelTutor is a comprehensive learning platform for OCaml, built to enhance your experience with the Real World OCaml book. It adds powerful interactive features to the official website:

  • Interactive Editors: Write and execute OCaml code with textbook-compliant remote code execution.
  • Site-wide Search: Use Cmd + K to quickly find topics and examples.
  • Embedded Practice: Engage with multiple-choice questions and hands-on programming exercises directly in the content.

CamelTutor integrates seamlessly with the official Real World OCaml website, ensuring the content is always up to date while adding these valuable tools.
